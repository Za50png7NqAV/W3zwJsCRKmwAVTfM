Education-Wise Details

Code Eligibility Criteria
NEET UG Code 01 Candidates appearing in qualifying examination (12th Standard) in 2025:
  • Can apply and take the test
  • Not eligible for MBBS/BDS admission if required pass percentage is not achieved before first round of Counseling
NEET UG Code 02 Higher/Senior Secondary Examination or Indian School Certificate Examination (10+2):
  • 12 years of study
  • Last 2 years must include Physics, Chemistry, Biology/Bio-technology (with practicals)
  • Mathematics or other elective subjects
  • English at core course level
NEET UG Code 03 Intermediate/Pre-degree Examination in Science:
  • Physics, Chemistry, Biology/Bio-technology (with practicals)
  • English as compulsory subject
NEET UG Code 04 Pre-professional/Pre-medical Examination:
  • After Higher Secondary/Pre-University or equivalent
  • Physics, Chemistry, Biology/Bio-technology (with practicals)
  • English as compulsory subject
NEET UG Code 05 First year of three years' degree course:
  • University Examination with Physics, Chemistry, Biology/Bio-technology (with practicals)
  • Must have passed earlier qualifying exam with same subjects
  • English at core course level or higher
NEET UG Code 06 B.Sc. Examination:
  • Minimum two subjects from Physics, Chemistry, Biology/Bio-technology
  • Must have passed earlier qualifying exam with Physics, Chemistry, Biology, and English
NEET UG Code 07 Any equivalent examination:
  • Last 2 years of 10+2 study must include Physics, Chemistry, Biology/Biotechnology
  • Must include practical tests in these subjects
  • Must include English
  • Must be equivalent to Intermediate Science Examination standard
